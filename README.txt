# Exécution du script SQL

exécuter le script "script-projet.sql" dans la DB créée pour les labos BDR avec les credentials suivant :
  - nom du schéma : projet
  - utilisateur : bdr
  - mdp : bdr

### Autres infos :

Le script à exécuter pour construire notre DB contient :
  - La création du schéma
  - La créatoin des tables et de leurs attributs ainsi que les contraintes
  - Le peuplement de la DB avec des infos fictives 
  - La création des vues
  - La création des triggers
